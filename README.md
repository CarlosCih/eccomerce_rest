# DRF
Proyecto de uso de REST en Django

## Descripción
Proyecto de eccomerce usando Django REST Framework para conocer el funcionamiento de APIs RESTful. Este proyecto esta basado en un tutorial de YouTube. El creador del tutorial es "@Developerpe", su link del canal es: https://www.youtube.com/@Developerpe.
El link de la playlist del tutorial es:
https://www.youtube.com/playlist?list=PLMbRqrU_kvbRI4PgSzgbh8XPEwC1RNj8F

## Instalación
1. Clonar el repositorio:
    git clone https://github.com/CarlosCih/eccomerce_rest.git
2. Crear un entorno virtual:
    python -m venv env
3. Activar el entorno virtual:
    - En Windows:
        .\env\Scripts\activate
    - En macOS/Linux:
        source env/bin/activate
4. Instalar las dependencias:
    pip install -r requirements.txt
5. Realizar las migraciones:
    python manage.py migrate
6. Iniciar el servidor de desarrollo:
    python manage.py runserver
7. Acceder a la aplicación en el navegador:
    http://127.0.0.1:8000/
